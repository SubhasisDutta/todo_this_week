📖 Help Guide

📋 Weekly Task Manager Overview

The Weekly Task Manager is a Chrome extension that helps you organize your work using a time-blocking methodology. Capture tasks, prioritize them, and schedule them into specific time slots in your week.

┌──────────────────────────────────────────────────┐ │ WEEKLY TASK PLANNER │ ├──────────┬──────────────────────────┬────────────┤ │Unassigned│ Weekly Grid (7 days) │ Assigned │ │ Tasks │ Mon Tue Wed Thu Fri │ Tasks │ │ ├─────┬─────┬─────┬─────┬──┤ │ │ [Task A] │ 🔥 │ │ ⭐ │ │ │ [Task B] │ │ [Task C] │ │ 🗓️ │ │ │ │ [Task D] │ │ │ │ │ │ │ │ │ └──────────┴─────┴─────┴─────┴─────┴──┴────────────┘

Key Concepts:

  • 🔥 CRITICAL — Urgent tasks with a deadline. Always visible at top.
  • ⭐ IMPORTANT — High-value tasks you should do this week.
  • 🗓️ SOMEDAY — Backlog items with no fixed timeline.
  • 🏠 Home / 💼 Work — Category for filtering.
  • 🍃 Low / ⚡ High energy — Match tasks to your energy levels.
  • 🔄 Recurring — Tasks that auto-renew daily/weekly/monthly.

Schedule Tab Features:

  • 🧩 Parking Lot — Collapsible sidebar for unscheduled tasks. Drag to grid to schedule.
  • 🤏 Snap Guides — Visual row/column guides when dragging tasks for precise placement.
  • 🕵️ Hover Details — Hover over grid tasks to see full details without opening edit form.
  • 🌗 Time Indicator — Red line shows current time. Past blocks are dimmed.

🚀 Quick Start Workflow

1

Add your tasks

Click the ➕ Add New Task button in the header (available from any tab), fill in the title, priority, type, and energy. Click Add Task. Or use the popup's ADD tab for quick capture.

2

Schedule tasks to time blocks

In the SCHEDULE tab, drag tasks from the left sidebar onto the grid cells. Each column is a day; each row is a time block.

3

Work through your day

Open the popup and go to TODAY tab to see tasks scheduled for today, grouped by time block. Check them off as you complete them.

4

Review completed tasks

Visit the ARCHIVE tab to see all completed tasks grouped by date. You can restore tasks or clear them.

5

Review your progress

Check the STATS tab for completion rates, priority distribution, and productivity charts.

💡 Tip: Press Ctrl+Z to undo the last destructive action (delete, complete, unassign all).

🗓️ Schedule Tab

The Schedule tab is the heart of the planner — a weekly grid of days × time blocks.

──────────┬──────┬──────┬──────┬──────┬────── Time | Mon | Tue | Wed | Thu | Fri ──────────┼──────┼──────┼──────┼──────┼────── [1PM-3PM] │ 🔥 │ │ ⭐ │ │ Deep Work │Task A│ │Task C│ │ ──────────┼──────┼──────┼──────┼──────┼────── [3PM-6PM] │ │ 🗓️ │ │ │ Deep Wk 2 │ │Task B│ │ │
1

Drag to schedule

Drag a task from the Unassigned sidebar onto any grid cell to schedule it for that day and time block.

2

Use the schedule form

Click ⏱️ on a task to open a scheduling grid — select multiple day/block combinations at once.

3

Slot limits

Some blocks (marked grey) are blocked (Sleep, Morning Prep, Lunch) and cannot receive tasks. Deep Work blocks accept only 1 task.

4

Unassign tasks

Drag a task from the grid back to the Unassigned sidebar to remove its schedule. Use Unassign All to clear the entire week.

💡 Today's column is highlighted in blue. Tasks in the Assigned sidebar can be expanded to see their individual schedule items and check them off.

⭐ Priority Tab

The Priority tab shows all your tasks organized in three columns by priority level.

Reorder tasks

Use the ↑ ↓ buttons to change the display order within a priority group.

2

Edit tasks inline

Click Edit to open an inline editing form. Changes auto-save after 1.5 seconds, or click Save immediately.

3

Search tasks

Use the search bar at the top to filter tasks by title. The filter is case-insensitive and updates as you type.

💡 CRITICAL tasks show days remaining or overdue. A task with all schedule items completed will automatically be marked complete.

🌍 Location Tab

The Location tab separates tasks into Home and Work categories, letting you focus on the right context.

┌─────────────────┬─────────────────┐ │ 🏠 HOME │ 💼 WORK │ ├─────────────────┼─────────────────┤ │ 🔥 Grocery list │ 🔥 Q4 Report │ │ ⭐ Read book │ ⭐ Team meeting │ │ 🗓️ Organize desk│ 🗓️ Update docs │ └─────────────────┴─────────────────┘

Within each column, tasks are sorted by priority (Critical → Important → Someday) then by display order.

Use the search bar at the top to quickly find tasks by title across both columns.

Use the Edit button to change a task's type (Home ↔ Work) — it will move to the correct column after saving.

🕐 Time Blocks

Click the Time Blocks button in the SCHEDULE tab to configure time blocks.

1

Manage Time Blocks

📦 Import / Export

Click the 📦 icon in the header to open the Import/Export modal. The modal is organized into four tabs for different import/export methods.

1

JSON Tab

Export all tasks to a JSON file for backup, or import tasks from a previously exported JSON file. The import merges with existing tasks: matching IDs are updated; new IDs are added. This is the recommended backup method.

2

CSV Tab

Import tasks from a CSV file. Required column: title. Optional columns: priority, type, energy, url, deadline, notes, recurrence. The tab shows format requirements and valid values for each column.

3

Google Sheets Tab

Import from a published Google Sheet. First publish your sheet as CSV (File → Share → Publish to web → CSV format). Then paste the URL, click Preview Sheet to verify the data, and click Import Rows.

4

Notion Tab

Connect to a Notion database for bidirectional sync. Enter your Integration Token and Database ID, fetch the schema, map your Notion properties to task fields, then use Sync Now for two-way sync or Import New Only to pull new items.

Import Logic (all methods): ┌─────────────────────────────────────────┐ │ Existing task ID → UPDATE (merge) │ │ New task ID → APPEND (add) │ │ No duplicates → Safe to re-import │ └─────────────────────────────────────────┘
💡 Use the JSON tab for regular backups. CSV and Google Sheets are best for bulk imports from spreadsheets. Notion sync keeps your tasks in sync with your Notion workspace.

🚀 Advanced Features (v1.5.0)

These powerful features help you work more efficiently with your schedule.

1

Magic Fill ✨

Click the Magic Fill button in the schedule header to automatically fill empty time blocks with quick tasks like "Check email" or "Stretch break". Works on today's schedule only. Great for filling awkward gaps between meetings.

2

Right-Click Context Menu

Right-click any task on the schedule grid to access quick actions:

  • Duplicate — Create a copy of the task
  • Color Code — Assign a custom color for visual organization
  • Split Task — Extend the task into the next time block
  • View Details — Open the task details modal
3

Focus Mode 🎯

Click the Focus Mode button to hide all time blocks except the current one. This reduces visual clutter and helps you concentrate on what's happening now. Click again to restore the full view.

4

Task Details (Double-Click)

Double-click any task in the Parking Lot or Assigned sidebar to open a detailed view showing all task information including notes, schedule, and URL. From there you can edit or delete the task.

5

Fluid Resizing

Drag the bottom edge of a task in the grid to extend it into subsequent time blocks. If another task is in the way, it will be automatically pushed down to make room.

6

Time Tracking ⏱️

Click the clock icon on any grid task to start tracking time. Click again when done to record the actual duration. This helps you compare planned vs. actual time for better future estimates.

7

Buffer Zones

When you have back-to-back tasks, a "5m buffer" indicator appears between them as a visual reminder to take a short break and transition mentally.

💡 All actions support Undo (Ctrl+Z). The hover popover shows task details when you hover over grid tasks for 400ms.

❓ Frequently Asked Questions

What happens to tasks scheduled for today that aren't completed?

Tasks that are scheduled but not completed remain on your schedule. They will show as incomplete in the TODAY tab of the popup. The schedule does not automatically roll over to the next day — you need to manually reschedule uncompleted tasks by dragging them to a new day in the SCHEDULE tab, or simply check them off when done.

How do I move a task to tomorrow?

In the SCHEDULE tab, find the task either on the grid or in the Assigned sidebar. Drag it to the desired day column. Alternatively, click the task to open its inline edit form, scroll to the Schedule section, and select a different day/time block combination.

What's the difference between completing a scheduled assignment vs. completing the task itself?

A task can have multiple schedule assignments (e.g., "Work on report" scheduled for Monday 1PM-3PM and Tuesday 1PM-3PM). You can check off each assignment individually. When ALL assignments for a task are completed, the task itself is automatically marked complete. In the popup Display tab, clicking the main checkbox completes the task and ALL its assignments at once.

How do recurring tasks work?

When you complete a recurring task, a new instance is automatically created with the same title, priority, type, and notes. The deadline (if set) advances by the recurrence interval: +1 day for daily, +7 days for weekly, or +1 month for monthly. The new instance starts with an empty schedule — you need to schedule it for the next period.

Can I undo a deleted task?

Yes! After deleting a task, an "Undo" toast appears briefly. Click it to restore the task. You can also press Ctrl+Z (or Cmd+Z on Mac) to undo the last destructive action. The undo history stores up to 5 states.

Why can't I drag a task to certain time blocks?

Some time blocks have slot limits. "Blocked" slots (like Sleep, Morning Prep, Lunch) don't accept any tasks. "Single" slots accept only one task per day. Check the Time Blocks configuration in the SCHEDULE tab to see each block's limit.

How do I back up my tasks?

Click the Import/Export button (📦) in the header, then click "Export Tasks". This downloads a JSON file with all your tasks. To restore, use "Import Tasks" and select your backup file. The import merges with existing tasks — matching IDs are updated, new IDs are added.

Why did my changes disappear after editing?

Inline edits auto-save after 1.5 seconds of inactivity. If you close the edit form or navigate away immediately, changes may not be saved. Watch for the "Saved" indicator, or click the Save button to save immediately.

What is the Parking Lot sidebar?

The Parking Lot (left sidebar in SCHEDULE tab) holds all unscheduled tasks. You can collapse it using the arrow button to gain more screen space for the grid. The badge shows how many tasks are waiting to be scheduled. Drag tasks from here onto the grid to assign them to time blocks.

What do the colored guide lines mean when dragging tasks?

When you drag a task over the schedule grid, blue guide lines highlight the target row (time block) and column (day) to help you place it accurately. Valid drop zones show a blue glow, while blocked or full slots show a red indicator with a "no entry" symbol.

How do I see full task details on the schedule grid?

Hover over any task on the grid for about half a second. A popover will appear showing the full title, notes, energy level, task type, deadline, URL, and schedule details. This helps you see task information without opening the edit form.

What does the red line on the schedule mean?

The red horizontal line shows the current time position on today's column. It updates every minute. Past time blocks (above the line) appear dimmed to help you focus on upcoming tasks. The current time block has a subtle red highlight.

CSV Format Requirements

  • Required column: title
  • Optional columns: priority, type, energy, url, deadline, notes, recurrence
  • Priority values: CRITICAL, IMPORTANT, SOMEDAY
  • Type values: home, work
  • Energy values: low, high
  • Recurrence values: daily, weekly, monthly
