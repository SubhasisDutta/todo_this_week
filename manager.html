📖 Help Guide

📋 Weekly Task Manager Overview

The Weekly Task Manager is a Chrome extension that helps you organize your work using a time-blocking methodology. Capture tasks, prioritize them, and schedule them into specific time slots in your week.

┌──────────────────────────────────────────────────┐ │ WEEKLY TASK PLANNER │ ├──────────┬──────────────────────────┬────────────┤ │Unassigned│ Weekly Grid (7 days) │ Assigned │ │ Tasks │ Mon Tue Wed Thu Fri │ Tasks │ │ ├─────┬─────┬─────┬─────┬──┤ │ │ [Task A] │ 🔥 │ │ ⭐ │ │ │ [Task B] │ │ [Task C] │ │ 🗓️ │ │ │ │ [Task D] │ │ │ │ │ │ │ │ │ └──────────┴─────┴─────┴─────┴─────┴──┴────────────┘

Key Concepts:

  • 🔥 CRITICAL — Urgent tasks with a deadline. Always visible at top.
  • ⭐ IMPORTANT — High-value tasks you should do this week.
  • 🗓️ SOMEDAY — Backlog items with no fixed timeline.
  • 🏠 Home / 💼 Work — Category for filtering.
  • 🍃 Low / ⚡ Medium / 🔋 High energy — Match tasks to your energy levels.
  • 🔄 Recurring — Tasks and events that auto-renew daily/weekly/monthly.
  • 📌 Event Notes — Lightweight appointments/errands with no completion status. Auto-expire when the time block passes.
  • 🌟 MIT Star — Mark one task or event per day as your Most Important Thing. Track streaks in Stats.
  • 📊 Groups Tab — View tasks grouped by any enabled attribute with bento grid visualization.

Schedule Tab Features:

  • 🧩 Parking Lot — Collapsible sidebar for unscheduled tasks and events. Drag to grid to schedule.
  • 📌 Event Notes — Add events via the "Add Event" button. Visually distinct from tasks with purple dashed borders.
  • 🌟 MIT Stars — Star any grid item as your Most Important Thing for the day. Exactly 1 per day.
  • 🤏 Snap Guides — Visual row/column guides when dragging tasks for precise placement.
  • 🕵️ Hover Details — Hover over grid tasks to see full details without opening edit form.
  • 🌗 Time Indicator — Red line shows current time. Past blocks are dimmed.

🚀 Quick Start Workflow

1

Add your tasks

Click the ➕ Add New Task button in the header (available from any tab), fill in the title, priority, type, and energy. Click Add Task. Or use the popup's ADD tab for quick capture.

2

Add events for appointments

Click 📌 Add Event in the SCHEDULE tab to create lightweight event notes for appointments, errands, or meetings. Events appear in the Parking Lot with a purple style.

3

Schedule tasks & events to time blocks

In the SCHEDULE tab, drag tasks and events from the Parking Lot sidebar onto the grid cells. Each column is a day; each row is a time block.

4

Star your Most Important Thing

Click the ☆ star on any grid item to mark it as your MIT for the day. Only 1 per day — this keeps you focused on what matters most.

5

Work through your day

Open the popup and go to TODAY tab to see tasks scheduled for today, grouped by time block. Check them off as you complete them. Events auto-expire when their time passes.

6

Review completed tasks

Visit the ARCHIVE tab to see all completed tasks grouped by date. You can restore tasks or clear them.

7

Review your progress

Check the STATS tab for completion rates, priority distribution, MIT streak, event stats, and productivity charts.

💡 Tip: Press Ctrl+Z to undo the last destructive action (delete, complete, unassign all). When you reopen the planner, a check-in modal will ask about past unresolved MITs.

🗓️ Schedule Tab

The Schedule tab is the heart of the planner — a weekly grid of days × time blocks. Schedule both tasks and events into your week.

──────────┬──────┬──────┬──────┬──────┬────── Time | Mon | Tue | Wed | Thu | Fri ──────────┼──────┼──────┼──────┼──────┼────── [1PM-3PM] │ 🔥⭐ │ │ 📌 │ │ Deep Work │Task A│ │Event │ │ ──────────┼──────┼──────┼──────┼──────┼────── [3PM-6PM] │ │ 🗓️ │ │ │ Deep Wk 2 │ │Task B│ │ │
1

Drag to schedule

Drag a task or event from the Parking Lot sidebar onto any grid cell to schedule it for that day and time block. Events appear below tasks in the sidebar, separated by a purple line.

2

Add events

Click 📌 Add Event to create event notes for appointments and errands. Events have a distinct purple dashed style, no completion status, and auto-expire when their time block passes. Recurring events regenerate automatically.

3

Star your MIT

Click the ☆ star on any grid item to mark it as your Most Important Thing for that day. Only 1 per day is allowed. The starred item gets a golden glow. Track your MIT completion in the Stats tab.

4

Slot limits

Some blocks (marked grey) are blocked (Sleep, Morning Prep, Lunch) and cannot receive tasks or events. Deep Work blocks accept only 1 item.

5

Unassign items

Drag a task from the grid back to the Parking Lot sidebar to remove its schedule. Use Unassign All to clear the entire week.

💡 Today's column is highlighted in blue. Tasks in the Assigned sidebar can be expanded to see their individual schedule items and check them off. Events auto-expire — no need to check them off.

📊 Groups Tab

The Groups tab provides a visual overview of your tasks organized by configurable attributes using a bento grid layout.

Bento Grid View

Each box shows an attribute with a mini donut chart showing the distribution of values. Click any box to drill down.

2

Drill-Down View

Clicking a bento box opens a detailed view with tasks grouped by each attribute value in columns. The view maximizes with a smooth slide animation. Use the search bar to filter tasks.

3

Task Interaction

Double-click any task to open the Task Details modal. From there you can view all task attributes as colorful tags, and use the Edit or Delete buttons. This provides a consistent experience across the app.

4

Configure Attributes

In Settings, toggle which attributes appear in the Groups tab. All 13 attributes can be enabled or disabled. Priority, Location, and Energy are enabled by default.

💡 Only enabled attributes appear in the bento grid and Task Details modal. Double-click tasks for quick access to edit/delete.

⚙️ Settings

Click the ⚙️ icon in the top-right corner to open Settings. All settings auto-save immediately when changed.

1

Dark / Light Mode

Toggle the theme switch. Changes apply immediately and are automatically saved.

2

Font Family & Size

Choose from 5 font families and 3 sizes. Changes apply instantly and are auto-saved.

3

Task Attributes

Toggle which task attributes are enabled. All 13 attributes can be toggled. Enabled attributes appear in: Groups tab bento grid, task forms, and Task Details modal. Priority, Location, and Energy are enabled by default. Disabled attributes are hidden from UI but data is preserved.

🕐 Time Blocks

Click the Time Blocks button in the SCHEDULE tab to configure time blocks.

1

Manage Time Blocks

View, add, or delete time blocks in the weekly grid. Click + Add Block and fill in the label, times, slot limit, and color. Use Reset to Defaults to restore the original configuration.

📦 Import / Export

Click the 📦 icon in the header to open the Import/Export modal. The modal is organized into four tabs for different import/export methods.

1

JSON Tab

Export all tasks to a JSON file for backup, or import tasks from a previously exported JSON file. The import merges with existing tasks: matching IDs are updated; new IDs are added. This is the recommended backup method.

2

CSV Tab

Import tasks from a CSV file. Required column: title. Optional columns: priority, type, energy, url, deadline, notes, recurrence. The tab shows format requirements and valid values for each column.

3

Google Sheets Tab

Import from a published Google Sheet. First publish your sheet as CSV (File → Share → Publish to web → CSV format). Then paste the URL, click Preview Sheet to verify the data, and click Import Rows.

4

Notion Tab

Connect to a Notion database for bidirectional sync. Enter your Integration Token and Database ID, fetch the schema, map your Notion properties to task fields, then use Sync Now for two-way sync or Import New Only to pull new items.

Import Logic (all methods): ┌─────────────────────────────────────────┐ │ Existing task ID → UPDATE (merge) │ │ New task ID → APPEND (add) │ │ No duplicates → Safe to re-import │ └─────────────────────────────────────────┘
💡 Use the JSON tab for regular backups. CSV and Google Sheets are best for bulk imports from spreadsheets. Notion sync keeps your tasks in sync with your Notion workspace.

🚀 Advanced Features

These powerful features help you work more efficiently with your schedule.

1

Magic Fill ✨

Click the Magic Fill button in the schedule header to automatically fill empty time blocks with quick tasks like "Check email" or "Stretch break". Works on today's schedule only. Great for filling awkward gaps between meetings.

2

Right-Click Context Menu

Right-click any task on the schedule grid to access quick actions:

  • Duplicate — Create a copy of the task
  • Color Code — Assign a custom color for visual organization
  • Split Task — Extend the task into the next time block
  • View Details — Open the task details modal
3

Focus Mode 🎯

Click the Focus Mode button to hide all time blocks except the current one. This reduces visual clutter and helps you concentrate on what's happening now. Click again to restore the full view.

4

Task Details (Double-Click)

Double-click any task in the Parking Lot or Assigned sidebar to open a detailed view showing all task information including notes, schedule, and URL. From there you can edit or delete the task.

5

Fluid Resizing

Drag the bottom edge of a task in the grid to extend it into subsequent time blocks. If another task is in the way, it will be automatically pushed down to make room.

6

Time Tracking ⏱️

Click the clock icon on any grid task to start tracking time. Click again when done to record the actual duration. This helps you compare planned vs. actual time for better future estimates.

7

Buffer Zones

When you have back-to-back tasks, a "5m buffer" indicator appears between them as a visual reminder to take a short break and transition mentally.

8

Event Notes 📌

Click the Add Event button to create lightweight event notes (appointments, errands) that are visually distinct from tasks. Events have no completion status — they auto-expire when their assigned time block passes. Recurring events auto-regenerate in the Parking Lot. Events can be duplicated across multiple time blocks.

9

MIT Star ⭐ (Most Important Thing)

Click the star icon on any grid task or event to mark it as your Most Important Thing for that day. Only 1 MIT per day is allowed — choosing a new one replaces the previous. When you open the planner with past unresolved MITs, a check-in modal asks if you completed them. Track your MIT streak and completion rate in the Stats tab.

💡 All actions support Undo (Ctrl+Z). The hover popover shows task details when you hover over grid tasks for 400ms.

❓ Frequently Asked Questions

What happens to tasks scheduled for today that aren't completed?

Tasks that are scheduled but not completed remain on your schedule. They will show as incomplete in the TODAY tab of the popup. The schedule does not automatically roll over to the next day — you need to manually reschedule uncompleted tasks by dragging them to a new day in the SCHEDULE tab, or simply check them off when done.

How do I move a task to tomorrow?

In the SCHEDULE tab, find the task either on the grid or in the Assigned sidebar. Drag it to the desired day column. Alternatively, click the task to open its inline edit form, scroll to the Schedule section, and select a different day/time block combination.

What's the difference between completing a scheduled assignment vs. completing the task itself?

A task can have multiple schedule assignments (e.g., "Work on report" scheduled for Monday 1PM-3PM and Tuesday 1PM-3PM). You can check off each assignment individually. When ALL assignments for a task are completed, the task itself is automatically marked complete. In the popup Display tab, clicking the main checkbox completes the task and ALL its assignments at once.

How do recurring tasks work?

When you complete a recurring task, a new instance is automatically created with the same title, priority, type, and notes. The deadline (if set) advances by the recurrence interval: +1 day for daily, +7 days for weekly, or +1 month for monthly. The new instance starts with an empty schedule — you need to schedule it for the next period.

Can I undo a deleted task?

Yes! After deleting a task, an "Undo" toast appears briefly. Click it to restore the task. You can also press Ctrl+Z (or Cmd+Z on Mac) to undo the last destructive action. The undo history stores up to 5 states.

Why can't I drag a task to certain time blocks?

Some time blocks have slot limits. "Blocked" slots (like Sleep, Morning Prep, Lunch) don't accept any tasks. "Single" slots accept only one task per day. Check the Time Blocks configuration in the SCHEDULE tab to see each block's limit.

How do I back up my tasks?

Click the Import/Export button (📦) in the header, then click "Export Tasks". This downloads a JSON file with all your tasks. To restore, use "Import Tasks" and select your backup file. The import merges with existing tasks — matching IDs are updated, new IDs are added.

Why did my changes disappear after editing?

Inline edits auto-save after 1.5 seconds of inactivity. If you close the edit form or navigate away immediately, changes may not be saved. Watch for the "Saved" indicator, or click the Save button to save immediately.

What is the Parking Lot sidebar?

The Parking Lot (left sidebar in SCHEDULE tab) holds all unscheduled tasks. You can collapse it using the arrow button to gain more screen space for the grid. The badge shows how many tasks are waiting to be scheduled. Drag tasks from here onto the grid to assign them to time blocks.

What do the colored guide lines mean when dragging tasks?

When you drag a task over the schedule grid, blue guide lines highlight the target row (time block) and column (day) to help you place it accurately. Valid drop zones show a blue glow, while blocked or full slots show a red indicator with a "no entry" symbol.

How do I see full task details on the schedule grid?

Hover over any task on the grid for about half a second. A popover will appear showing the full title, notes, energy level, task type, deadline, URL, and schedule details. This helps you see task information without opening the edit form.

What does the red line on the schedule mean?

The red horizontal line shows the current time position on today's column. It updates every minute. Past time blocks (above the line) appear dimmed to help you focus on upcoming tasks. The current time block has a subtle red highlight.

What is the Groups tab?

The Groups tab shows your tasks grouped by enabled attributes (Priority, Type, Energy, etc.) in a bento grid layout. Each box shows the task count and a mini donut chart. Click any box to drill down and see tasks grouped by that attribute's values. Double-click any task to open the Task Details modal.

How do I view all details for a task?

Double-click any task in the Parking Lot, Assigned sidebar, or Groups drill-down to open the Task Details modal. It shows all task attributes as colorful tag badges, plus notes, URL, and schedule. Click "Edit" to modify the task or "Delete" to remove it.

Does the task edit modal auto-save?

Yes! When editing a task via the Task Details modal's Edit button, changes auto-save after about 1 second of inactivity. The button shows "Saving..." then "Saved ✓" to confirm. The Groups tab and other views update immediately.

Which task attributes does Notion import support?

Notion import supports 10 task attributes: title, priority, type, energy, deadline, notes, URL, impact, value, complexity, action, estimates, and interval. Configure column mapping in the Notion tab of the Import/Export modal to match your Notion database properties.

How does the status attribute work with the completion checkbox?

The status and completion checkbox are synchronized. When you check the completion checkbox, the status is set to "Done". When you uncheck it, the status resets to "Inbox". Conversely, if you set the status to "Done" or "Archive" (via the edit form), the checkbox appears checked. This bidirectional sync ensures consistency between workflow status and completion state.

What's the difference between Done and Archive status?

Both "Done" and "Archive" are completion states. "Done" means the task was completed successfully. "Archive" is for tasks that were dropped, cancelled, or set aside. Both appear in the Archive tab. Checking the completion checkbox sets status to "Done", but if you manually set status to "Archive", it stays "Archive" when you check the box.

How does Notion auto-mapping work?

When you fetch your Notion database schema, the extension automatically maps select field values that match (case-insensitive) between local values and Notion values. For example, if your Notion has "Critical" and the local option is "CRITICAL", they're automatically mapped. You can always manually adjust mappings after auto-mapping.

What are Event Notes and how are they different from tasks?

Event Notes are lightweight items for appointments and errands (e.g., "Go to Costco", "Visit friend"). Unlike tasks, events have no completion status or priority. They auto-expire when their assigned time block passes. Recurring events regenerate in the Parking Lot. Events appear with a distinct purple/dashed style so you can tell them apart from tasks at a glance.

What happens when an event expires?

When the time block an event is assigned to passes (based on the actual date and time), the event is automatically removed from the grid. For recurring events, a fresh instance is created in the Parking Lot so you can schedule it for next time. Non-recurring events are deleted entirely.

What is MIT (Most Important Thing)?

MIT lets you mark exactly one task or event per day as your Most Important Thing by clicking the star icon. This helps you focus on what matters most. The Stats tab tracks your MIT completion streak and rate. When you open the planner with past unresolved MITs, a check-in modal asks if you completed them.

Can I have more than one MIT per day?

No, MIT is intentionally limited to exactly one per day. This forces you to identify the single most important thing. If you star a different item, it replaces the previous MIT for that day. This constraint is a key part of the productivity methodology.

How do I edit an event?

Double-click any event in the Parking Lot, on the grid, or in the Assigned sidebar to open the Edit Event modal. You can change the title, notes, recurrence, and color. Changes auto-save when you close the modal. To delete an event, click the Delete button in the edit modal.

What do the event colors mean?

Event colors are customizable categories. By default, they're named by color (Red, Blue, Green, etc.). Go to Settings → Event Color Labels to rename them based on your workflow (e.g., "Meetings", "Personal", "Health"). The Stats tab shows color distribution based on your labels.

Why are tasks in the Parking Lot ordered a certain way?

Tasks in the Parking Lot are sorted by last modified time, with the most recently modified tasks at the top. This helps you quickly find tasks you've been working on. Events appear after tasks, separated by a dashed line.

How do I know if my task edits are saved?

When editing a task, look for the auto-save indicator in the modal header. It shows "Saving..." while saving and "✓ Auto-saved" when complete. Changes are saved automatically after about 1 second of inactivity, so you don't need to click a Save button.

📄 JSON Import / Export

Export your tasks to a JSON file for backup, or import tasks from a previously exported file. The JSON format preserves all task data including schedules, notes, and settings.

⬇️ Export Tasks

Download all your tasks as a JSON file. This creates a complete backup that can be imported later.

Import tasks from a JSON file. Existing tasks with matching IDs will be updated; new tasks will be added.

📊 CSV File Import

Upload a CSV file with your tasks. This is useful for importing tasks from spreadsheets or other task managers.

CSV Format Requirements

  • Required column: title
  • Optional columns: priority, type, energy, url, deadline, notes, recurrence
  • Priority values: CRITICAL, IMPORTANT, SOMEDAY
  • Type values: home, work
  • Energy values: low, high
  • Recurrence values: daily, weekly, monthly
